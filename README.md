# Проект Mesto (бэкенд)

Бэкенд-часть самостоятельной практической работы, выполненной в рамках обучения на курсе [Веб-разработчик](https://praktikum.yandex.ru/web) от [Яндекс.Практикум](https://praktikum.yandex.ru) на спринтах №10-15.
